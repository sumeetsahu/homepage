# Profile Image

The current profile image (`profile.svg`) is a placeholder with your initial "S".

## To add your actual profile photo:

1. Add your photo to the `public/` folder (e.g., `profile.jpg`)
2. Update `src/data/profile.json`:
   ```json
   "profileImage": "/profile.jpg"
   ```

Recommended:
- Use a square image (e.g., 400x400px)
- Formats: JPG, PNG, or WebP
- Keep file size under 500KB for fast loading
